App name: Photo Manager
Simple system that allows user to register for them to store their photos by creating different albums
Allows them to save multiple images per album as well as delete images within the album


Requirement for use:
*must have xampp
*run apache
*run mysql
*internet is needed for bootstrap

How to setup:
Copy the folder named photomanager from SourceCode folder to htdocs
Create database named photomanager_db in phpmyadmin 
Then import photomanager_db.sql in the the said databse

access system by going to:

localhost/photomanager/public
